## PageAnalyzer

Web application that executes queries over the network and saves data to a database.

### Requirements

- PHP: >=7.4
- Composer: 2.3
- Postgresql: 12+

### Setup

$ git clone git@github.com:CheshirSmil/php-project-9.git

$ cd php-project-9

$ make install

$ make start
